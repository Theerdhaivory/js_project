The console

The JavaScript console is a command line interface in your browser that can execute snippets of code.

When that code snippet is designed to interact with the webpage you are currently on, result can happen that might not have been possible otherwise.

how to Open the console

Up to date directions can be found online for opening your browser's JavaScript console, but short descriptions follow below for certain browsers as of the time of writing of this article: Firefox: Menu Bar > Tools > Web Developer > Web Console Chrome: View> Developer>Developer Tools Safari: Safari Menu > Preferences > Advanced > Checkbox "Show Develop Menu" > Develop Menu > Show Error Console
